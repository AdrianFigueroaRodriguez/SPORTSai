# SPORTSai

This repository contains a single-page HTML presentation for the **ARF Totals Model — NBA Analytics Engine**.

## Live view

Run a local static server from the repo root and open the page in a browser:

```bash
python3 -m http.server 4173
```

Then visit:

- `http://127.0.0.1:4173/arf-totals-model.html`
