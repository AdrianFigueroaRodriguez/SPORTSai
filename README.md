# Adversarial Regime Analysis Wireframe

A lightweight wireframe prototype that turns a sports betting analysis prompt into interactive UI controls and structured output tables.

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## What you can tune

- Games list
- Bankroll + ticket count
- Home advantage
- Probability uncertainty standard deviation (σₚ)
- Hidden chaos factor
- Regime weight sliders
- Allocation override toggles (extreme skew, injury shock, CLV underperformance)

## Generated outputs

- Regime classification
- Probability + σₚ table
- Capital allocation table
- Scenario coverage map
- Risk commentary
