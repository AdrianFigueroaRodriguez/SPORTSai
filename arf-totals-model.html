Model Architecture

Core Variables

CORE-01
Pace Alignment Coefficient
PAC
Measures possession volume and tempo agreement between both teams. Alignment factor of 1.10 indicates strong tempo agreement; 0.90 indicates conflicting styles.
18% – 26%
CORE-02
Market Edge Delta
MED
Projected Total minus Market Total. Zones: <2 noise, 2–4 tradable, 4–6 high edge, >6 revalidation required. Linear scale capped at 8 pts.
20% – 28%
CORE-03
Shooting Convergence Index
SCI
Measures offensive efficiency compatibility via 2P% cross-matching plus 3P volume interaction coefficient. Scaled 0–100.
14% – 20%
CORE-04
Free Throw Interaction Index
FTI
Cross-product of FT rate and foul% for each team. Adjusted by spread proximity and late-game script probability. Scaled 0–100.
12% – 18%
CORE-05
Spread Stability Score
SSS
Encodes game script volatility. Spread <5 scores 100; 6–9 scores 80; 10+ scores 60. Tighter spreads increase confidence in scoring exchange.
8% – 14%
SEC-06
ORTG Compression Index
OCI
100 minus the absolute ORTG gap. High compression improves scoring exchange persistence. Monte Carlo calibrated secondary variable.
5% – 12%
SEC-07
Defensive 3P Compatibility
D3PC
3P attempt rate times opponent 3P allowed %. Overs spike when high 3P volume meets permissive perimeter defense. Monte Carlo calibrated.
5% – 12%

Mathematical Framework

Model Formula

// CONFIDENCE SCORE FUNCTION
CONF = w1 × PAC + w2 × MED + w3 × SCI + w4 × FTI + w5 × SSS + w6 × OCI + w7 × D3PC

// CONSTRAINTS
Σ(w1..w7) = 100%
Σ(w1..w5) ≥ 75% // Core minimum
Σ(w6..w7) ≤ 25% // Secondary cap

// OPTIMIZATION TARGET
F = (0.5 × HitRate≥80%) + (0.3 × EV) + (0.2 × StabilityScore)
// Subject to: Drawdown < 12%
